# Security Policy

如果你发现的是安全问题，请不要直接公开提 issue。

更合适的方式：

- 发邮件到：`happysnaker@foxmail.com`
- 标题建议：`[qq-ai-bot security] 简短问题描述`

## 请优先私下报告的情况

- access token / credentials 泄漏
- 未授权访问
- 可能导致远程命令执行的问题
- 会话隔离失效
- 敏感信息错误暴露

## 报告时尽量包含

- 影响范围
- 复现步骤
- 最小配置片段（请脱敏）
- 日志或报错信息
- 你认为的风险等级

## 响应原则

我会尽量按这个顺序处理：

1. 确认是否能复现
2. 判断影响范围
3. 优先准备修复
4. 在合适时间公开修复说明

## 支持版本

当前优先处理最新公开版本与 `main` 分支上的安全问题。
